# Teacher-Grade-Report

# Description

This C++ program helps a teacher analyze the scores of 10 students using arrays and loops.

# Features

* Stores 10 student scores in an array
* Calculates the sum of all scores
* Calculates the average score
* Counts how many students passed (70 or higher)
* Finds the highest score
* Finds the lowest score

# Concepts Used

* Arrays
* For loops
* Conditional statements
* Variables
* std::fixed and std::setprecision
